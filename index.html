Surgimento da internet

(Voltar)

A internet surgiu como um projeto militar chamado ARPANET nos anos 1960, desenvolvido pela Agência de Projetos de Pesquisa Avançada (ARPA) dos Estados Unidos. Inicialmente concebida como uma rede de comunicação resistente a falhas em caso de ataques nucleares durante a Guerra Fria, a ARPANET evoluiu para se tornar a base do Protocolo de Controle de Transmissão/Protocolo de Internet (TCP/IP).

A década de 1990 testemunhou a disseminação comercial da internet, que se expandiu globalmente e transformou-se na World Wide Web (WWW), criada por Tim Berners-Lee em 1989. Desde então, a internet se tornou uma ferramenta essencial para comunicação, comércio e entretenimento em todo o mundo.

Como funcina o HTML

(Voltar)

o HTML é utilizado para estruturar e formatar o conteúdo de páginas web através de tags, que serão interpretadas pelo navegador. Para que seja possível essa interpretação, é importante declarar qual a versão do HTML será usada.

Essa declaração é feita através do , garantindo que o navegador interprete corretamente o código. É importante observar que isso não é uma tag HTML, e sim uma instrução para o navegador que deve ser colocada no início do documento sem espaços entre ela.

O que são Tags

(Voltar)

Em HTML, as "tags" são elementos fundamentais que definem a estrutura e o significado do conteúdo em uma página da web. Elas são compostas por palavras-chave ou siglas envolvidas por colchetes angulares ("<" e ">"). As tags HTML são usadas para marcar diferentes partes do conteúdo, indicando ao navegador como deve ser apresentado ou interpretado.

Os princípios básicos para fazer um site em HTML fazem parte da seguinte estrutura inicial:

  • <html> responsável por definir o início e fim do documento HTML;
  • <head> define as informações do documento;
  • <title> é o título da página;
  • <body> é onde constará todo conteúdo visível da página.

Elementos em linha

(Voltar)

Elementos em linha, também conhecidos como inline, são elementos que ocupam somente a largura do seu conteúdo, como as tags:

  • <a>: é possível definir links com essa tag;
  • <span>: é um container genérico para agrupar textos;
  • <br>: cria uma quebra de linha;
